# 羅喉傳：天都戰記 v5

## 遊戲玩法

### 人物移動方式
- 玩家透過點擊「地圖按鈕」移動
- 每個區域代表一個場景（廢墟 / 荒原 / 古都 / Boss區）
- 點擊後會顯示：
  - 目前位置
  - 簡單劇情描述

### 小敵人系統
- 荒原、古都會隨機遭遇敵人
- 敵人類型：
  - 掠奪者
  - 鬼智師
- 擊敗敵人：
  - 獲得道具
  - 提升等級
  - 解鎖下一區域

### Boss 遭遇方式
- 需要先擊敗「荒原 + 古都」敵人
- 解鎖最終區域（深淵核心）
- 進入後觸發 Boss：
  - 魔化將軍（多技能）

### 戰鬥系統
- 普通攻擊：基礎輸出 + 累積怒氣
- 武君戰印：消耗 MP，高傷害
- 計略觀心：回復 MP
- 計都斬：滿怒氣才能使用

### 背包與裝備
- 點擊即可使用或裝備
- 裝備影響角色能力

### 系統功能
- 存檔 / 讀檔
- 重新開始
