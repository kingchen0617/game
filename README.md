# 羅喉傳：天都戰記 v3 強化版

這是基於原始 `kingchen0617/game` 專案整理的可直接部署版本，保留原本的劇情節點與回合制戰鬥，並補上以下可玩性強化：

## 本次補強內容
- 自動存檔：推進節點與戰鬥勝利後自動保存
- 標題畫面續玩：若偵測到存檔會顯示「繼續遊戲」並標示時間
- 劇情打字跳過：點擊劇情文字即可完整顯示
- 戰鬥防禦架勢：降低下一回合傷害並回少量真元
- 軍糧丹系統：可在戰鬥中補血，部分劇情與首領會提供
- 敵方意圖顯示：更清楚預判戰鬥節奏
- 戰後獎勵與戰績：勝利時會獲得補給或資源
- 結局總結：顯示命運傾向與整體通關數據

## 執行方式
直接開啟 `index.html` 即可。
