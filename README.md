# 羅喉傳：天都戰記

一款以布袋戲角色「羅喉」為靈感改編的 Web RPG 原型遊戲。

## 專案特色

- 純前端 HTML / CSS / JavaScript
- 可直接在瀏覽器開啟遊玩
- 劇情選擇 + 回合制戰鬥
- 角色狀態、技能、敵人 AI 與多結局雛形
- 適合作為 GitHub Pages 部署起點

## 遊戲概念

玩家將扮演羅喉，在天都崛起、江湖征戰與王道抉擇之間前行。你需要：

- 在劇情節點中做出抉擇
- 與敵方勢力交戰
- 管理生命、真元與怒氣
- 走向霸道、王道或犧牲之路

## 執行方式

直接開啟 `index.html` 即可。

## 建議後續擴充

- 地圖探索系統

- 更多角色與敵人
- 音效與動態轉場
