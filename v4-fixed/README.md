# 羅喉傳：天都戰記 v4-fixed

這是目前可運行的修正版 Web RPG 原型，包含：

- 地圖探索
- 回合制戰鬥
- 背包與裝備
- 掉寶
- 存檔 / 讀檔

## 遊戲操作方式

### 開始遊戲
- 進入 `v4-fixed/index.html`
- 點擊 **開始遊戲** 進入主畫面
- 點擊 **操作說明** 可查看玩法

### 地圖探索
- 進入遊戲後，畫面中會顯示地圖區域
- 點擊已解鎖區域即可前往探索
- 若該區域有敵人，會自動進入戰鬥

### 戰鬥操作
戰鬥中可使用以下按鈕：

- **普通攻擊**：造成基礎傷害，並累積怒氣
- **武君戰印**：消耗真元，造成較高傷害
- **計略觀心**：恢復真元
- **終式・計都斬**：怒氣達到 100 後可使用

### 背包與裝備
- 背包中的 **消耗品**：點擊即可使用
- 背包中的 **裝備**：點擊即可裝備到對應欄位
- 裝備會影響角色數值：
  - 武器增加攻擊
  - 防具增加氣血上限
  - 飾品增加真元上限

### 系統功能
- **儲存進度**：將目前遊戲狀態存到瀏覽器
- **讀取進度**：讀取已儲存的遊戲
- **重新開始**：重置遊戲進度
- **音效**：目前為預留功能
